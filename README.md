# Help Desk  
Учебный проект на Java 17, Spring Boot, Spring MVC и Thymeleaf.  
## Возможности 
- главная страница сайта 
- страница "О нас" 
- страница "Контакты поддержки"  
## Запуск 
1. Установить JDK 17.
2. Запустить проект:    Windows: mvnw.cmd spring-boot:run    macOS/Linux: ./mvnw spring-boot:run 
3. Открыть в браузере:    http://localhost:8080
